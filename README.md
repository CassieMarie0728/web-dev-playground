#  Web Dev Playground (React + Vite + Tailwind)

This repo is your front-end sandbox.

**Use with Codex environment:** `Web Dev Badass Starter Kit (React + Vite + Tailwind)`

### What this repo is for

- Learning React
- Playing with Vite
- Experimenting with TailwindCSS
- Building throwaway components and pages

### How to use it with Codex

1. Open this repo in Codex.
2. Select the `Web Dev Badass Starter Kit` environment.
3. Tell Codex something like:
   - *"Scaffold a new Vite + React + Tailwind project in this repo."*
   - *"Create a basic landing page component with my brand colors."*
4. Let Codex create the project structure and install dependencies.
5. Run the dev server with `npm run dev` or `pnpm dev` (depending on what Codex set up).

### .gitignore

This repo includes a Node-friendly `.gitignore` so build files and dependencies don't pollute your commits.
